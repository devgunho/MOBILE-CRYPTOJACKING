DeepSound - Stenography Tool for Windows
Review by George Norman on 08 Nov, 2012
With the advanced stenography tool DeepSound installed on your Windows-powered machine, you will be able to hide data into audio files and extract hidden data from audio files. If you want to, you can encrypt the hidden data with the reliable AES 256-bit encryption algorithm.

DeepSound works on any Windows-powered PC that has the .NET Framework installed on it. To get DeepSound up and running, you must go through a standard setup process: download a small installer, run it, click through the instructions presented to you by a setup wizard. In little time and without any hang-ups, you will be done with the installation process and you will be able to start using DeepSound.

The application’s interface is divided into two panes. To the left you have the “Carrier audio files” pane that allows you to select the carrier files that will feature hidden data and select the carrier files that already have hidden data inside them. To the right there’s a “Secret files” pane that displays the data hidden into audio files.

With DeepSound, hiding data into an audio file is a simple 3-step process:
Step 1 – select the carrier file; choose a WAV or FLAC audio file.
Step 2 – select the file you want to hide.
Step 3 – hit Encode, choose the output directory, choose if you want to encrypt and password protect the files, then let DeepSound do its job.

DeepSound is a free application. Consider making a donation if you enjoy using it and want to support its development.

Pros
It takes little time to install the application and get accustomed with the interface. Use DeepSound to encode data into WAV and FLAC audio files; use DeepSound to extract hidden data from WAV and FLAC audio files. There’s an option to encrypt and password protect files using the AES 256-bit algorithm. DeepSound is freeware.

Cons
None that I could think of.
