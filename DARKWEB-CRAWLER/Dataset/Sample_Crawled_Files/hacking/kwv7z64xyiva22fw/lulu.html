About OWASP

The Open Web Application Security Project (OWASP) is a 501(c)(3) worldwide not-for-profit charitable organization focused on improving the security of software. Our mission is to make software security visible, so that individuals and organizations worldwide can make informed decisions about true software security risks.

These books are all available to buy "at cost". OWASP earns no revenue from their sale. All books are also available to download free of charge as source files or PDFs from the OWASP website at https://www.owasp.org.

Please consider buying extra copies and donating them to local groups, clubs, libraries, schools, colleges and universities.
