Security in a Box - Digital security tools and tactics

The Tactics Guides in this toolkit cover basic principles, including advice on how to use social media and mobile phones more safely. The Tool Guides offer step-by-step instructions to help you install, configure and use some essential digital security software and services.

The Community Toolkits focus on specific groups of people â sometimes in specific regions â who face significant digital security threats. They include tailored advice on tools and tactics that are relevant to the needs of these particular groups.

Security in-a-Box is jointly developed by Front Line Defenders and Tactical Technology Collective, along with a global network of thousands of activists, trainers and digital security experts.
