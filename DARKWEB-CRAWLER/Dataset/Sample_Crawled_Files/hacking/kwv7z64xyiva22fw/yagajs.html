YAGA leaflet-ng2

Angular module for Leaflet, which serves components and directives for Leaflet in Angular (2 and above).

  • leaflet-ng2

    Granular Leaflet integration for Angular (2 and above)
    GitHub npm Docs Project Server Examples Unit-Tests Test-Coverage

  • leaflet-ng2-gps

    GPS service and directive for leaflet-ng2
    GitHub npm

  • leaflet-ng2-custom-control

    Custom control for leaflet-ng2
    (unpublished)

  • ionic2-starter-leaflet-ng2

    Ionic starter template for leaflet-ng2
    GitHub

YAGA docker

Docker container for geo spatial purposes.

  • docker-geoserver

    Docker files for GeoServer container.
    GitHub docker

  • docker-mapproxy

    Docker files for mapproxy container.
    GitHub docker

  • docker-solr

    A full spatial-ready version of the official docker solr image.
    GitHub docker

  • docker-postgis

    This docker image is an enhanced version of the original postgres docker image.
    GitHub docker

  • docker-osmosis

    Docker files for Osmosis container.
    GitHub docker
