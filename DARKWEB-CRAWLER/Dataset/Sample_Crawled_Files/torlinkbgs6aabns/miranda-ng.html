Miranda OTR
Filename MirOTR.dll
Authors ProgAndy, CHEF-KOCH
Download links
Stable version: 32-bit, 64-bit
Development version: 32-bit, 64-bit

OTR (Off-the-Record) plugin for Miranda NG (using some code and ideas from SecureIM, Pidgin-OTR and old Miranda OTR (by SJE)).

Receiving OTR encrypted messages

To read OTR encrypted messages the recipient needs either Miranda with this plugin or any of the following clients/plugins:

  • Adium (built-in support)
  • ChatSecure (Jabber, Android)
  • Climm (former mICQ) (built-in support)
  • Jitsi (multi-platform)
  • MCabber (built-in support)
  • OTR plugin for Pidgin
  • Psi+ (built-in support)
  • OTR plugin for Trillian
  • OTR plugin for Trillian (new project)
  • OTR plugin for Kopete (KDE)
