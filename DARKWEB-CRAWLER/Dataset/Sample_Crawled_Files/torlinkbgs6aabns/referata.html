WikiPapers

WikiPapers is a compilation of resources (conference papers, journal articles, theses, books, datasets and tools) focused on the research of wikis. It aims to create the most comprehensive literature compilation for this research area. WikiPapers was started in April 2011 by Emilio J. How can I help?

We are searching for volunteers. You can help filling missing fields in pages using the "Edit" tab or the [+] links. The following two categories include pages needing completion: Category:Publication template parameter issues and Category:Author template parameter issues.

Also, you can create new pages and you are more than welcome to add your research publications regarding wikis. Finally, read WikiPapers, it is fun!

How can I create a new page?

For adding new content to WikiPapers, you can use the following forms:

  • Form:Publication: books, conference papers, journal articles, theses, etc.
  • Form:Author: researchers.
  • Form:Event: conferences, workshops, notable meetings, etc.
  • Form:Keyword: keywords.
  • Form:Dataset: datasets and corpuses.
  • Form:Journal: journals.
  • Form:Tool: tools and software.

Export

You can export WikiPapers metadata in the following formats: BibTeX, CSV, RDF and JSON. Use the links available in the infoboxes. There is also an API which you can use to explore the wiki in an automatic way.

Download

You can download a XML dump with the full histories of this wiki. If this site vanishes in the future, nothing will be lost. Keep a copy on your computer. We use and recommend WikiTeam tools.
